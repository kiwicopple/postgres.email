# `postgres.email`

A website to display all Postgres emails.

## Background

This is a small POC to transform the Postgres [mailing lists](https://www.postgresql.org/list/) more readable.

As a comparison:

- ["[PATCH] btree merge scan proposal" on postgres.email](https://postgres.email/lists/pgsql-hackers/AS1PR02MB78468B976C3198FD926CEA039A9FA%40AS1PR02MB7846.eurprd02.prod.outlook.com)
- ["[PATCH] btree merge scan proposal?" on postgres list](https://www.postgresql.org/message-id/AS1PR02MB78468B976C3198FD926CEA039A9FA%40AS1PR02MB7846.eurprd02.prod.outlook.com)

The idea of postgres.email is to make all responses threaded, like the comments on HN or Reddit.
